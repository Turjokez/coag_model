DS/shear ratio map check notes

This file records what was checked in the ratio maps.

1) Quantity plotted:
log10(beta_DS / beta_shear)

Note:
If both terms are weighted by the same C1*C2, that factor cancels.
So map pattern is the same for kernel ratio and weighted ratio.

2) Kernel mode used:
direct_sinking_law

3) Color meaning:
- negative value -> shear > DS -> blue
- positive value -> DS > shear -> red

4) Main result:
When epsilon increases, DS-dominant area gets smaller.
This is physically expected.

Shape note:
- current law shows a narrow V-shape near diagonal
- literature laws show a wider band near diagonal
- this comes from different sinking-law slope behavior
